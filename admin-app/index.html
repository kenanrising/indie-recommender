API & Adalo Setup
Everything you need to connect your Adalo app to the recommender.
Step 1 — Export your catalog
1
Add all your tracks using the Add Track form or the CSV import.
Then click Export CSV to download your complete catalog file.
2
Export your CSV — this is the file your backend needs.
You'll upload this file to your Render deployment.
Step 2 — Deploy the backend to Render
1
Go to GitHub.com and create a free account if you don't have one.
2
Create a new repository called indie-recommender and upload all the files from the zip you downloaded.
3
Go to Render.com, create a free account, click New → Web Service, and connect your GitHub repo. Render auto-reads the render.yaml file — no configuration needed.
4
Upload your catalog CSV to the data/ folder in your GitHub repo (replace tracks_sample.csv with your exported file, renamed to tracks.csv).
5
Run the embedding pipeline from the Render Shell tab:
python scripts/build_embeddings.py --csv data/tracks.csv --out embeddings/
Step 3 — Wire to Adalo Custom Action
https://YOUR-APP.onrender.com/recommend_deep
POST
{ "track_id": "<Magic Text — Current Track ID>", "k": 10 }
{ "results": [ { "track_id": "TRK004", ← bind to "Recommended Track ID" "title": "Low Signal", ← bind to "Track Title" "artist": "Mara Blue", ← bind to "Artist Name" "score": 0.912 ← bind to "Similarity Score" } ] }
